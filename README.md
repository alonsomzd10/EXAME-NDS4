# Sistema de Gestión del Zoológico

EXAMEN para la administración de datos de animales mediante archivos CSV.
Proyecto de desarrollo en Python que permite listar animales por clase o características utilizando archivos CSV.

## Cómo ejecutar el programa

1. Abrir la carpeta zoologico en VS Code
2. Ejecutar el archivo main.py

## Instrucciones de uso
1. Asegúrese de tener los archivos `zoo.csv` y `clases.csv` en la carpeta raíz.
2. Ejecute el programa desde la terminal con: `python main.py`
3. Utilice el menú interactivo para filtrar animales o agregar nuevos registros.
